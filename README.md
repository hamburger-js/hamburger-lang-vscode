# hamburger-lang README

a DSL highlight for hamburger-lang.

```text

// example
VStack {             @padding=10 @alignItems=center
  Image(imgLink)     @theme=thumbnail
  Text {             @variant=secondary
    Text(name)       @bold
    Text(email)
  }
Button("Follow")     @theme=primary
}
```
